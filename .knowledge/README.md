# Project Knowledge Base

This directory contains shared knowledge files for the project.
These files are managed by `lk` (local-knowledge-cli) and synced to a local SQLite DB.

## Structure
- `architecture/` - System design and architecture knowledge
- `features/` - Feature-specific knowledge
- `conventions/` - Coding conventions and patterns

## Format
See https://github.com/xxx/local-knowledge-cli for the markdown format specification.
